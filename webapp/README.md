# Webapp Frontend

Aplicação Next.js simples para enviar mensagens ao endpoint `/message` da API dotnet-app.

## Como rodar localmente

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Acesse em http://localhost:3000

## Configuração do endpoint

O endpoint está configurado como `http://localhost/message` no código. Altere para o endereço público da API se necessário.
